# 🛡️ WorkerGuard (워커가드)

---

## 서비스 이름
**WorkerGuard (워커가드)**  
물류센터 인력·안전 관리를 웹에서 한 번에 관리할 수 있는 대시보드

---

## 서비스 소개

WorkerGuard는 물류센터에서 자주 발생하는 **엑셀 기반 인력 관리의 비효율**,  
그리고 **과로 위험 확인의 어려움**을 빠르게 해결하기 위해 제작한 해커톤 프로젝트입니다.

정규직/일용직을 나누어 관리하고, 근무 기록을 업로드하면  
바로 급여·위험군·피로도 등 핵심 지표를 자동으로 계산해주는 **경량화된 통합 관리 툴**입니다.

---

## 서비스 핵심 기능

### 🚨 위험군 자동 탐지  
- 최근 근무 강도 기반 위험군 자동 표시  
- 전일/당일 피로도 비교  
- 휴식 권고 시나리오 제공

### 💰 정규직 / 일용직 급여 정산  
- 정규직: 월 누적, 야간수당 자동 반영  
- 일용직: D-3 지급 로직 구현  
- 개인별 급여 명세서 제공

### 📄 엑셀 기반 데이터 업로드  
- 명단 및 근무기록 업로드  
- 중복 업로드 방지  
- 즉시 테이블 반영

### 🔄 직무 비율 기반 업무 배치  
- 직무 비율 설정  
- 자격증 기반 필터  
- 자동 배정 시뮬레이션

### 🔐 권한 분리  
- 관리자 / 스태프 권한 구분  
- 비밀번호 bcrypt 해시 저장
